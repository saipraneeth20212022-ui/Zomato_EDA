# Zomato Restaurant Data Analysis (EDA)

Exploratory Data Analysis on a restaurant dataset to uncover patterns in ratings, votes, costs, and online ordering behavior across restaurant types.

## 📊 Overview
This project analyzes 148 restaurants to understand:
- Which restaurant types are most popular (by votes)
- How ratings vary across restaurant categories
- Whether online ordering correlates with better ratings
- Cost trends across restaurant types

## 🛠️ Tools Used
- Python
- Pandas
- Seaborn / Matplotlib
- Plotly (interactive visualizations)
- Jupyter Notebook

## 🔍 Key Steps
1. Data cleaning & type conversion
2. Null value & data quality checks
3. Grouping & aggregation (votes, ratings, cost by type)
4. Visualization (bar charts, box plots)
5. Comparative analysis: Online vs Offline order ratings

## 📈 Key Insights
- **Dining** is the most popular restaurant type by total votes
- Restaurants offering **online ordering have higher median ratings (~3.9)** compared to those that don't (~3.4)
- **Buffet** restaurants are the most expensive on average (₹671 for two), while **Dining** is the most budget-friendly (₹357 for two)
- "Other" category restaurants also tend to be premium-priced (₹669 avg)
